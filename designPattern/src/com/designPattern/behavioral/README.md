## 解决什么问题

行为型模式分解对象间的职责和行为，将对象间的通信解耦，以更灵活的控制。
这里额外说一下，面向对象是把行为和对象绑定，但行为模式又是对行为解耦，说明面向对象不是什么都好，最起码在某些方面耦合关系太强。
